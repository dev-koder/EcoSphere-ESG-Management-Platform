# EcoSphere ESG Management Platform

An open-source ESG management platform built on Odoo.

## Setup
1. Run `docker compose up -d`
2. Open `localhost:8069` in your browser.
3. Create a database named `ecosphere`. Do NOT check the demo data box (we ship our own).
4. Go to Apps, remove the "Apps" filter, find "EcoSphere: ESG Management" and click Install.
